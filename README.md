# MYTH NUMBER GAME

## About
Game yang menghadirkan konsep dimana dalam hitungan 5 detik setiap mulai acak akan muncul angka misterius secara acak dan pemain harus menebak angak tersebut dengan cepat karena angka tersebut akan hilang dengan cepat.

## MODE PERMAINAN
**Live Mode**

Memainkan langsung secara offline dan bisa dilakukan minimal 2 orang atau lebih dengan tekan tombol mulai acak. Jika pemain berhasil menebak angka dengan benar, maka host wajib mengacak angka lagi dengan memilih tombol Acak Lagi.

**Play Mode**

Main langsung secara mandiri dan tebak angka dengan benar dengan tekan tombol mulai acak. Pemain bisa memilih Ulangi jika tidak yakin dengan jawabannya atau salah menjawab.
